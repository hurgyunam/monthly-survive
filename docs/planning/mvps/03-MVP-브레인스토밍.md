# MVP 쪼개기 전 브레인스토밍

별 생각 없이 떠오르는 생각들을 자유롭게 적어두는 공간입니다.  
정리하거나 우선순위를 정할 필요 없이, 일단 쭉 옮겨 적어보세요.

---

## 💭 떠오르는 생각들

- **장르:** 모바일 다인원 개척 생존 시뮬 (RimWorld + FM2020)
- **핵심:** 직접 컨트롤 불가 → 설정만 하고 한 달 관찰 → 재설정 루프
- 월 단위 사이클: 설정 → 시뮬 돌림 → 결과 보고 → 다시 설정
- 플레이어 = 지휘관/감독 역할 (직접 조작 X)
- **게임 변수:** 초반 지역을 플레이어가 고를 수 있게 한다. 지역별로 일어날 수 있는 자연 재해를 다르게 두어 변수를 준다. 랜덤하게 외부 세력이 약탈하러 오는 이벤트를 넣어 변수와 긴장감을 더한다.

## 📦 자원 리스트 (디테일)

| 자원 | 설명 / 용도 |
|------|-------------|
| **목재** | 나무 채집·벌목으로 획득. 건축, 연료, 가구·도구 제작의 기본 재료. |
| **흙** | 채굴·수확으로 획득. 농경지, 벽돌·도자기, 기반 시설 건설에 사용. |
| **석재** | 채석·채굴로 획득. 건축, 방어 시설, 고급 구조물의 재료. |
| **철광석** | 광산·채굴로 획득. 제련 후 철 → 무기, 도구, 기계 부품 제작. |
| **구리광석** | 광산·채굴로 획득. 제련 후 구리 → 전선, 전기 시설, 합금 재료. |
| **석탄** | 광산·채굴로 획득. 연료, 제련용 열원, 발전·가열에 사용. |
| **석유** | 유정·채굴로 획득. 연료, 화학 제품, 고급 에너지원. |
| **우라늄** | 희귀 광산·채굴로 획득. 원자력 발전, 고급 에너지·연구용. |

## 📝 나중에 추가로 적고 싶을 때

아래에 그때그때 덧붙이면 됩니다.

- **시설/건축물:** 조합대(제작대), 용광로(철광석→철괴), 가마(흙→도자기) 등 재료를 가공·제작할 수 있는 시설을 건설·운용할 수 있게 한다. 시설마다 필요한 자원·조건이 다르고, 특정 시설이 있어야만 고급 아이템이나 2차 가공품을 만들 수 있게 설계.
- **자원 조합 → 아이템:** 자원들을 적절하게 조합해 아이템을 만들 수 있게 한다. (예: 목재+철괴 → 도구, 흙+석탄 → 벽돌) 레시피/조합 규칙을 두어 어떤 자원 조합이 어떤 아이템이 되는지 정의하고, 시설·레벨 등 조건에 따라 해금되거나 제작 가능하도록 설계.
