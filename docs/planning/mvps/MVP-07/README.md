# MVP-07: 시설을 이용한 제작 + 제작 계획 UI

> **검증 목표:** "시설을 이용해 자원을 조합해 아이템을 제작하고, 제작 계획을 세울 수 있는 UI가 월 단위 생존과 잘 엮이는가?"  
> → [03-MVP-브레인스토밍.md](../03-MVP-브레인스토밍.md)의 **자원 조합 → 아이템** 항목 반영.

---

## 🎯 목표

- **시설을 이용한 제작**: 조합대, 용광로, 가마 등 **보유 시설**에서 자원을 조합해 **아이템**을 만들 수 있게 한다. (예: 철광석 → 용광로 → 철괴, 목재+철괴 → 조합대 → 도구) 레시피/조합 규칙을 두고, 시설·조건에 따라 제작 가능하도록 한다.
- **제작 계획 UI**: 플레이어가 **무엇을 얼마나 만들지** 계획을 설정할 수 있는 UI를 제공한다. (예: 이번 달 철괴 10개, 도구 2개)

"수집 → 시설 건설 → **가공·제작 계획** → **제작**"이 월간 생존 루프와 연결되도록 한다. (필수 기획: [docs/planning/필수-기획-사항.md](../../필수-기획-사항.md) 제작 기획 참고)

---

## 📦 범위 (In Scope)

### 1. 레시피·제작 정의
- **레시피** 정의: 어떤 자원 조합 → 어떤 아이템 (예: 철광석 → 용광로 → 철괴)
- **시설별 제작 가능 목록** — 조합대에서 도구, 용광로에서 철괴 등
- 제작 시 **필요 자원·소요 시간** 반영 (필수 기획 반영)
- 월 시뮬 중 "제작 지시" 또는 자동 가공 규칙에 따라 자원→아이템 전환

### 2. 제작 계획 UI
- 월 초(또는 사이클 시작 전) **제작 계획** 설정 UI
  - 제작할 아이템(레시피) 선택, 목표량·우선순위 설정
  - "이번 달 목표: 철괴 10개, 도구 2개" 등
- 설정된 계획이 시뮬/결과에 반영 — 결과 화면에서 제작 완료 여부·진행 상황 표시

### 3. 월 시뮬·결과 연동
- 제작된 아이템이 **자원/생존**에 반영 (예: 도구 보유 시 채집 효율 상승 — 선택)
- 결과 화면에 제작 이력 또는 요약 표시

### 4. 데이터/상태
- 레시피(재료, 결과물, 필요 시설, 제작 시간) 데이터
- 제작 계획(대상 레시피, 목표량/우선순위) 저장

## ✅ 완료 조건

- [ ] **보유 시설**을 이용해 **자원 조합(레시피)**으로 아이템을 **제작**할 수 있고, 제작이 자원·시간과 연동된다.
- [ ] **제작 계획 UI**로 플레이어가 제작할 아이템·목표량/우선순위를 설정할 수 있고, 설정이 시뮬/결과와 연동된다.
- [ ] 시설·제작이 월 단위 생존 루프와 연결되어 "수집→시설→제작 계획→제작"이 한 사이클로 느껴진다.
