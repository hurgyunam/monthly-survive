# 필수 기획 사항

> **목적:** 본격 개발/확장 전에 반드시 정해야 할 기획 항목을 명시합니다.  
> **대상:** 기획자, PM, 개발 참여자

---

## 1. 게임의 최종 목표

**게임 자체의 최종 목표를 반드시 정의해야 한다.**

- 플레이어가 "이겼다" 또는 "목표를 달성했다"고 느끼는 **승리 조건**이 무엇인지
- 혹은 엔딩/클리어 개념이 없는 무한 생존인지, 특정 시점/목표 달성 시 종료인지
- 이에 따라 월간 루프, 자원 설계, 난이도 곡선이 달라지므로 **우선적으로 확정**할 필요가 있음

→ 별도 문서(예: `design/최종목표.md`)에서 정의하고, MVP 기획 시 이 목표에 맞춰 단계별 목표를 쪼갤 것.

---

## 2. 시설 건설 기획

**시설 건설 시 필요한 자원과 소요 시간에 대한 기획이 필요하다.**

- 각 **시설(건물)별**
  - 건설에 **필요한 자원** (종류, 수량)
  - 건설에 **들어가는 시간** (게임 내 시간/일 단위 등)
- 자원 부족, 시간 제한에 따른 우선순위·밸런스 설계의 기반이 됨

→ 시설 목록이 확정되면 시설별 스펙 문서 또는 데이터 시트로 정리할 것.

### 시설 건설과 재미 요소

기획 시 아래와 같은 특징을 어떻게 잡느냐에 따라 플레이 감성이 달라진다.

| 특징 | 값을 높게/많게 잡을 때 | 값을 낮게/적게 잡을 때 |
|:---|:---|:---|
| **필요 자원 종류·수량** | 수집·계획의 재미, "언제 모을까" 선택의 긴장감 증가. 다만 과하면 수집 피로로 이어질 수 있음 | 빠르게 건설 가능 → 초반 피드백이 좋고 캐주얼해짐. 장기 목표감은 약해질 수 있음 |
| **건설 소요 시간** | 완성까지 기다리는 기대감, 중장기 목표와 성취감. 대기 구간이 길면 다른 활동(제작·수집)과 자연스럽게 엮임 | 즉시 완성에 가깝게 → 만족은 빠르지만 "다 짓고 나면 할 일 없음"이 빨리 올 수 있음 |
| **시설 간 의존성(선행 시설)** | "A를 지어야 B를 지을 수 있다" 같은 단계 구조 → 진행 욕구와 단계별 성취감. 로드맵이 분명해짐 | 시설을 거의 독립적으로 지을 수 있으면 자유도는 높지만, 목표가 흐트러질 수 있음 |
| **희소 자원 사용** | 특정 시설에만 쓰는 귀한 자원 → "이걸 어디에 쓸까" 우선순위 결정의 재미, 전략성 증가 | 자원이 흔하면 선택의 부담이 줄어들고 편하지만, 의미 있는 선택이 줄어들 수 있음 |

→ 목표하는 플레이 스타일(캐주얼 vs 심도 있는 계획, 단기 vs 장기 목표)에 맞춰 위 항목들의 균형을 정하는 것이 좋다.

## 3. 제작(크래프팅) 기획

**제작 시 필요한 자원과 소요 시간에 대한 기획이 필요하다.**

- 각 **제작 레시피별**
  - 제작에 **필요한 자원** (재료 종류, 수량)
  - 제작에 **들어가는 시간** (게임 내 시간/일 단위 등)
- 시설 건설·생존 루프와 연동되므로, 시설 기획과 함께 정리하는 것이 좋음

→ 제작 테이블/레시피가 확정되면 제작별 스펙 문서 또는 데이터 시트로 정리할 것.

### 제작(크래프팅)과 재미 요소

제작 기획의 특징에 따라 "수집·조합·대기"에서 오는 재미가 달라진다.

| 특징 | 값을 높게/복잡하게 잡을 때 | 값을 낮게/단순하게 잡을 때 |
|:---|:---|:---|
| **레시피 복잡도(재료 종류·수량)** | "이걸 모아서 이걸 만든다" 발견·연결의 재미, 레시피 해금·완성의 만족감. 다만 진입 장벽이 올라갈 수 있음 | 재료가 적고 단순 → 바로 만들어 쓰기 쉬움. 접근성은 좋고, 깊이는 줄어들 수 있음 |
| **제작 소요 시간** | 만드는 동안 다른 일(수집·시설 건설)을 하게 됨 → 게임 루프가 자연스럽게 엮이고, "완성됐다" 알림의 기쁨이 커짐 | 즉시 제작에 가깝게 → 즉각 만족은 크지만, 제작이 게임 흐름의 축이 되기 어려움 |
| **재료 수집 난이도** | 구하기 어려운 재료 → 구했을 때의 보상감, "이걸로 뭘 만들까" 선택의 무게감 | 쉽게 구할 수 있으면 제작이 부담 없지만, 도전감과 희소성은 줄어듦 |
| **시설·레시피 연동** | "이 시설에서만 이걸 만들 수 있다" → 시설 건설 목표와 제작 목표가 연결되어 장기 루프가 형성됨 | 시설과 무관하게 만들 수 있으면 유연하지만, 월간 생존 루프와의 결합이 약해질 수 있음 |

→ 시설 건설 기획과 맞춰 "수집 → 제작 → 시설/생존에 활용"이 한 사이클로 느껴지도록 설계하면, 월 단위 생존의 리듬과 목표감이 살아난다.
