## 🎯 MVP 쪼개기 체크리스트

각 항목을 추가하기 전에 확인:

- [ ] **독립 실행 가능**: 이 MVP만으로도 어떤 가치를 제공하는가?
- [ ] **검증 가능**: "성공했다"고 판단할 수 있는 기준이 있는가?
- [ ] **작은 범위**: 2주 안에 끝낼 수 있는 크기인가?
- [ ] **의존성 파악**: 어떤 MVP가 선행되어야 하는가?

---

## 📌 MVP 추가 템플릿

새 MVP를 추가할 때 아래 형식으로 `mvps/MVP-XX-이름.md` 파일을 생성하세요.

```markdown
# MVP-XX: [이름]

## 목표
(이 MVP로 검증/달성하고 싶은 것)

## 범위 (In Scope)
- [ ] 기능 1
- [ ] 기능 2

## 범위 밖 (Out of Scope)
- (의도적으로 제외한 것)

## 완료 조건
- [ ] 조건 1
- [ ] 조건 2
```
