## 1. MVP-02 추가/변경 컴포넌트

### 1.1 ColonistRoleCard (설정 화면용)

콜로니원 1명에 대한 역할 선택 카드.

| 속성 | 값 |
|:---|:---|
| 높이 | 88pt |
| 패딩 | 16pt |
| 배경 | `--bg-card` |
| border-radius | `--radius-md` |
| 콜로니원 간 간격 | 12pt |

**내부 구조 (림월드 스타일 우선순위 테이블):**
```
┌─────────────────────────────────────────────────────────┐
│ [아바타 40x40]  콜로니원 1                      [간격]  │
│ ┌──────────┬────────┬────────┬────────┐                │
│ │          │  채집  │  건축  │  제작  │  (컬럼 헤더)   │
│ ├──────────┼────────┼────────┼────────┤                │
│ │          │  [1]   │  [2]   │  [ ]   │  ← 셀: 숫자 또는 빈칸 │
│ │          │        │        │        │   좌클릭 감소, 우클릭 증가 │
│ └──────────┴────────┴────────┴────────┘                │
└─────────────────────────────────────────────────────────┘
```

- 우선순위 숫자: 1, 2, 3, 4
- 상단: 아바타 + 이름 ("콜로니원 1")
- 하단: **컬럼** = 채집 | 건축 | 제작. **셀** = 우선순위 숫자 또는 **0일 때 빈칸** (빈칸 = 해당 활동을 절대 하지 않음).

- **클릭 동작:** **좌클릭** = 숫자 감소, **우클릭** = 숫자 증가. 0이 되면 빈칸으로 표시. 우선순위 숫자는 모든 활동에 각각 배정 가능

**활동 선택:** 림월드 스타일 — 활동별 셀에 숫자(1~4) 또는 빈칸(0) 표시. 좌클릭으로 감소, 우클릭으로 증가. 0에서 좌클릭 할경우 4.

## 2. 화면별 변경 사항

### 2.1 설정 화면 (SetupScreen)

**기존:** 활동 1종 선택 (전체)

**변경:**
- "이번 달에 집중할 활동을 선택하세요" → "각 콜로니원의 역할을 선택하세요"
- ActivitySelectStack (3개) → ColonistRoleCard 2~3개 (스택)
- 각 ColonistRoleCard 내부: **림월드 스타일** — 컬럼(채집/건축/제작), 셀에 우선순위 숫자(1~3) 또는 빈칸(0). 좌클릭 감소·우클릭 증가로 수정

### 2.2 시뮬 화면 (SimulationScreen)

**기존:** ColonistCard 1개

**변경:**
- ColonistCard 2~3개 세로 스택
- 각 카드: 이름 + 배정된 활동 "~중..."

### 2.3 결과 화면 (ResultScreen)

**기존:** ResultCard 1개 (활동 + 수치)

**변경:**
- ResultColonistRow 2~3개 (또는 ResultCard 내부 리스트)
- 각 행: "이름 활동1: 수치1, 활동2: 수치2, ..." (예: "재현 채집: 48, 건축: 24, 제작: 12")
