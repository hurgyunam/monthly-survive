# MVP-06: 시설(조합대·용광로·가마 등) 건설 + 건설 계획 UI

> **검증 목표:** "시설을 건설하고, 어떤 시설을 언제 지을지 계획을 세울 수 있는 UI가 월 단위 생존과 잘 엮이는가?"  
> → [03-MVP-브레인스토밍.md](../03-MVP-브레인스토밍.md)의 **시설/건축물** 항목 반영.

---

## 🎯 목표

- **시설 건설**: 조합대(제작대), 용광로, 가마 등 **재료를 가공·제작하는 시설**을 건설할 수 있게 한다. 시설마다 필요한 자원·조건이 다르고, 건설 소요 시간이 반영된다.
- **건설 계획 UI**: 플레이어가 **어떤 시설을 언제 지을지** 계획을 설정할 수 있는 UI를 제공한다. (예: 이번 달 조합대, 다음 달 용광로)

"자원 수집 → 시설 건설 계획 → 건설"이 월간 생존 루프와 연결되도록 한다. (필수 기획: [docs/planning/필수-기획-사항.md](../../필수-기획-사항.md) 시설 건설 기획 참고)  
**제작(자원 조합→아이템)** 은 MVP-07에서 다룬다.

---

## 📦 범위 (In Scope)

### 1. 시설 정의·건설
- **시설 목록** 정의 (최소: 조합대, 용광로, 가마 — 필요 시 1~2종만 먼저)
- 각 시설별 **건설 필요 자원·소요 시간** (필수 기획 반영)
- **건설 UI** — 자원 보유 시 건설 가능, 완료 시 시설 보유 상태 저장
- (선택) 시설 간 선행 관계 (A 지어야 B 건설 가능)

### 2. 건설 계획 UI
- 월 초(또는 사이클 시작 전) **건설 계획** 설정 UI
  - 건설할 시설 종류 선택, 순서/우선순위 설정
  - "이번 달 목표: 조합대 완공" 등 목표 설정
- 설정된 계획이 시뮬/결과에 반영 — 결과 화면에서 건설 완료 여부·진행 상황 표시

### 3. 월 시뮬·결과 연동
- 보유 시설이 **월 시뮬 상태**에 반영 (MVP-07에서 해당 시설로 제작 시 사용)
- 결과 화면에 시설 건설 이력 또는 요약 표시

### 4. 데이터/상태
- 시설 정의(이름, 필요 자원, 건설 시간) 데이터
- 건설 계획(대상 시설, 순서/우선순위) 저장

---

## 🚫 범위 밖 (Out of Scope)

| 제외 항목 | 이유 |
|:---|:---|
| 시설을 이용한 제작(레시피·아이템) | MVP-07에서 다룸 |
| 모든 브레인스토밍 시설 | 우선 2~3종만 |
| 지역별 시설 제한 | 이후 확장 |
| 복잡한 해금 트리 | 단순 건설 가능 여부 수준만 |

---

## ✅ 완료 조건

- [ ] **조합대·용광로·가마** 중 최소 2종 이상 시설을 **건설**할 수 있고, 건설에 필요한 자원·시간이 반영된다.
- [ ] **건설 계획 UI**로 플레이어가 건설할 시설·순서/우선순위를 설정할 수 있고, 설정이 시뮬/결과와 연동된다.
- [ ] 시설 건설이 월 단위 생존 루프(자원 획득·소비)와 연결되어 "수집→계획→건설"이 한 사이클로 느껴진다.
